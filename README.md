# College Exam Seat Planner

## Objective
Allocate exam classrooms using the minimum number of rooms while preferring lower-floor classrooms.

## Features
- Add classroom details
- View classroom list
- Allocate exam seats
- Error handling for insufficient seats

## Tech Stack
- React (Vite)
- JavaScript
- Vercel

## Allocation Logic
- Greedy algorithm
- Lower floor classrooms prioritized
- Minimum number of rooms allocated
